# 🚖 Smart Ride

A full-stack ride booking and subscription management platform developed using the MERN stack. The application provides secure user authentication, premium subscription plans, Stripe payment integration, and an administrative dashboard for monitoring users and subscriptions.

---

## Overview

Smart Ride is designed to demonstrate modern full-stack web development practices. The system allows users to create accounts, manage subscriptions, access premium features, and interact with a secure payment workflow. Administrators can monitor platform activity through a dedicated dashboard.

---

## Features

### User Module

* User Registration and Login
* JWT-Based Authentication
* Protected Routes
* User Dashboard
* Subscription Management

### Premium Membership

* Multiple Subscription Plans
* Premium Account Status Tracking
* Subscription Activation and Expiry Management

### Payment System

* Stripe Payment Gateway Integration
* Secure Checkout Process
* Payment Success and Cancellation Handling

### Admin Module

* Dashboard Statistics
* User Management
* Subscription Monitoring
* Administrative Controls
